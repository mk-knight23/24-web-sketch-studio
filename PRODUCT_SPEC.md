# Product Specification - MK SketchFlow

- **Core Problem:** Users need a simple, fast, free, registration-free utility to solve: "Drawing, whiteboard and image-annotation studio".
- **Target Audience:** Everyday internet users, developers, and professionals.
- **Key Workflows:**
  1. Instant utility access on page load (simple first).
  2. Advanced configuration options via drawers/tooltips (progressive disclosure).
  3. Safe localStorage or IndexedDB persistence.
- **Features Included:** Pencil, brush, shapes, layer management, image import annotation, canvas export, auto-save.
